Getting started

We wanted to design RandDOS so it is as flexible as possible and when others host it that it wouldnt take hours to properly configure. Thats why the html document has been specially modified so it could run off one singular html file and nothing else! Essentially everyone can host a RandDOS instance just using a html file and not all of it. There will be a site made for RandDOS instances so all of the ones made will be shown (you will have to complete a form for yours to be acknowledged!)
